# AI Agent Fewer Decisions Efficiency

This example demonstrates the concept of how reducing the number of decisions an AI agent makes can lead to increased efficiency. It simulates two agents performing the same task: calculating the sum of positive numbers and the product of negative numbers in a list. One agent follows a less optimized strategy with more decision points, while the other employs a more strategic, fewer-decision approach, highlighting the benefits of streamlined decision-making.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
